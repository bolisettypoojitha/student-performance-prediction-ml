# student-performance-prediction-ml
# Hybrid Machine Learning Approach for Student Performance Prediction

This project predicts student academic performance using machine learning techniques.
A hybrid approach integrating SMOTE and feature selection is proposed to handle class
imbalance and feature redundancy.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Matplotlib
